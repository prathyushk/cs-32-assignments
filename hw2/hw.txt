The difference is a stack uses a depth first search while a queue is a breadth first search. This means for a stack, it will follow each path to the end and work backwards from there, following all of the paths to the end. For a queue, it follows one layer at a time in a breadth first search. It checks the surrounding places, then those that surround the surrounding places, etc. until it finds the end.
